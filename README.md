# ai-document-summarizer

An AI-powered web application that generates concise summaries from text-based documents using a simple and intuitive interface.

## Features

- Upload `.docx`, `.md`, and `.txt` documents
- Generate concise AI-powered summaries
- Fast and easy-to-use interface
- Responsive web design

## Supported File Types

- Microsoft Word (`.docx`)
- Markdown (`.md`)
- Plain Text (`.txt`)

> **Note:** PDF files are currently not supported.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- OpenAI API

## Getting Started

```bash
npm install
npm run dev
```
